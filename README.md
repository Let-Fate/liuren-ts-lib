# liuren-ts-lib

一个使用 TypeScript 编写的，基于 [tyme](https://github.com/6tail/tyme4j) 的六壬神课- TypeScript lib 库。

## ✨ 特性

-   六壬神课排盘
-   TypeScript 支持

## 📦 安装

```bash
yarn add liuren-ts-lib
```

## 🔨 使用

```typescript

```

## 📜 脚本

-   `yarn build`: 编译 TypeScript 代码。
-   `yarn dev`: 监听文件变化并自动编译。
-   `yarn test`: 运行测试。
-   `yarn lint`: 检查代码风格。
-   `yarn lint:fix`: 自动修复代码风格问题。
